# Website-for-Crimson-Bakery-using-HTML-CSS-JS-and-PHP-backend
A modern and responsive bakery website for Crimson Bakery built using HTML, CSS, JavaScript, and PHP with a functional backend.
1. Import the provided SQL file into your MySQL database using phpMyAdmin.
2. Download this project and place the folder inside: C:\xampp\htdocs\
3. Open XAMPP and start Apache + MySQL.
4. Visit http://localhost/myprojectfinal/Homeloged.php in your browser.
